# 006 Compiler First

ATLAS should not begin as a framework.

It should begin as a compiler.

Memory has nothing to preserve until understanding has been compiled.

Graph has nothing meaningful to connect until knowledge has been compiled.

Reasoning has nothing stable to use until Project DNA exists.

The first proof of ATLAS is:

```text
atlas compile
  ->
project.amf
```

## Principle

```text
Compile understanding first.

Build runtime later.
```
