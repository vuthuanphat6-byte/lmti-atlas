# Changelog

All notable changes to this project will be documented in this file.

This project has not published a stable release yet.

## Unreleased

### Added

- Local-first Project Atlas workflow through `lmti compile`.
- Context Pack generation through `lmti context`.
- Policy-safe preflight path through `lmti preflight`.
- Project memory, short memory, lesson capture, lifecycle, and retrieval flows.
- Privacy gates, redaction, egress scanning, and security-focused docs.
- Codex-first workflow documentation.
- Open-source contribution, security, roadmap, changelog, and issue templates.

### Notes

- Package distribution is not finalized.
- The verification metadata contract is documented, but exact
  `verify_required`, `suggested_verification`, and `last_verified_at` fields are
  not yet first-class on every context item.
