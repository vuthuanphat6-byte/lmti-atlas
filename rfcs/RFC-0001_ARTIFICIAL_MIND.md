# RFC-0001: Artificial Mind

Status: Draft

Canonical document: `docs/ATLAS-0001_ARTIFICIAL_MIND.md`

## Question

What is an Artificial Mind?

## Position

Artificial Mind is not memory, retrieval, graph storage or prompt history.

Artificial Mind is:

```text
Understanding
+
Knowledge
+
Memory
+
Reasoning
+
Evolution
+
Cognitive Privacy
+
Identity
```

## Design Consequence

ATLAS must not begin as a package tree.

ATLAS begins as a cognitive runtime whose modules exist only to preserve,
protect and evolve understanding.

## Review Notes

This RFC remains Draft until the surrounding Sprint 0 specifications define how
the Mind lifecycle is compiled, stored, reasoned over, protected and evolved.
