# RFC-0005: Cognitive Privacy

Status: Draft

## Question

How does ATLAS protect cognition as an asset?

## Scope

This RFC will define the privacy boundary for sensitive knowledge, memory,
relationships, reasoning traces and context packs.

## Required Design Pressure

The RFC must answer:

* What counts as sensitive cognition?
* What can leave ATLAS?
* What must remain encrypted?
* How are permissions represented?
* How are external models constrained?
* How are leaks detected or prevented?

## Non-Goals

* exposing raw confidential memory,
* treating knowledge like ordinary data,
* relying on model behavior as a security boundary.
