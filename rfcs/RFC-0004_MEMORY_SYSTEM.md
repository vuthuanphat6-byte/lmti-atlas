# RFC-0004: Memory System

Status: Draft

## Question

How does ATLAS remember meaning without becoming a memory database?

## Scope

This RFC will define long-term memory, short-term memory and memory-based
reasoning boundaries.

## Required Design Pressure

The RFC must answer:

* What is memory allowed to store?
* What is memory forbidden to store?
* How is memory connected to understanding?
* How are stale memories corrected?
* How are sensitive memories isolated?
* How is forgetting designed intentionally?

## Non-Goals

* prompt history storage,
* raw repository storage,
* unencrypted sensitive memory,
* vendor-specific memory features.
